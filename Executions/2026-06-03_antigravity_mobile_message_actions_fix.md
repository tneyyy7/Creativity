## Входные данные
Жалоба пользователя о том, что на мобильной версии сайта при клике на сообщение не появляется поле с выбором действий (меню контекстных действий).

## Результат
1. Изменен обработчик `onClick` в `src/pages/Messages.jsx` для пузырей сообщений. Теперь он срабатывает на мобильных устройствах и планшетах (`isMobileOrTablet`), открывая меню действий (`setSelectedContextMsg`).
2. Добавлена фильтрация кликов по интерактивным элементам (кнопки, ссылки, видео, изображения) для предотвращения нежелательного открытия меню при клике по ссылкам на профили, медиа и т.д.
3. Протестировано с помощью эмуляции мобильного устройства в браузере. Теперь при тапе на сообщения меню выбора действий корректно открывается, а при тапе на ссылки профилей открывается профиль пользователя.
